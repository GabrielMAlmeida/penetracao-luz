# 🌟 Simulação de Penetração de Luz em Tecidos Biológicos (Método Monte Carlo)

Aplicação web desenvolvida em Python para simular a penetração da luz em tecidos biológicos utilizando o método de Monte Carlo, permitindo comparar diferentes comprimentos de onda e propriedades ópticas dos tecidos.

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como ferramenta de apoio para uma pesquisa acadêmica na área de Odontologia.

A aplicação foi criada para auxiliar estudos relacionados à interação da luz com tecidos biológicos, oferecendo uma forma simples e visual de analisar diferentes cenários de propagação da luz.

Além disso, o sistema foi disponibilizado em um servidor online, permitindo seu acesso diretamente pelo navegador, sem a necessidade de instalação ou configuração local.

---

## 🧬 Funcionalidades

- Simulação da penetração de luz em tecidos biológicos;
- Comparação entre diferentes comprimentos de onda:

  - 660 nm
  - 808 nm
  - 904 nm

- Simulação em diferentes tipos de tecido:

  - Mucosa
  - Osso

- Ajuste dos parâmetros ópticos:

  - μa — Coeficiente de absorção;
  - μs — Coeficiente de espalhamento;
  - g — Fator de anisotropia;
  - n — Índice de refração.

- Visualização gráfica da profundidade de penetração da luz;
- Comparação entre diferentes configurações ópticas;
- Exportação dos resultados para análise posterior;
- Acesso online através de navegador web.
